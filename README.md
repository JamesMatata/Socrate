### Socrate: AI-Powered Study Partner
#### Overview
Socrate is an AI-powered web application designed to help students engage with their study materials. Users can upload documents, videos, and audio files, and interact with an AI that can summarize content, extract key entities, and hold live discussions based on the uploaded material.

#### Features
1. **Upload Materials**: Supports PDFs, Word documents, videos, and audio files.
2. **AI-Powered Chat**: Engage in live discussions with an AI based on the uploaded material.
3. **Text and Voice Interaction**: Communicate with the AI through text or voice messages.
4. **Material Management**: View, manage, and delete uploaded materials.
#### Tools and Technologies Used
1. **Django**: Web framework used to build the application.
2. **Django Channels**: Enables real-time communication via WebSockets.
3. **OpenAI API**: Powers the AI chat functionality.
4. **Google Cloud APIs**: Used for speech-to-text and vision-related tasks.
5. **PyMuPDF**: Extracts text from PDFs.
6. **MoviePy**: Handles video processing.
7. **Vosk**: Used for offline speech recognition.
8. **Spacy**: Performs entity extraction from text.
